# Zammie / 자미(姿美)
Turtle Neck Detection (Service Learning, Physical Computing Exhibition Project)
프로젝트 Zammie는 일상 속에서 종종 놓치고 지나치기 쉬운 우리의 자세와 행동에 대해 다시금 성찰하고 고민해 볼 수 있는 기회를 제공하는 것을 목표로 합니다. 이 프로젝트는 우리가 다양한 일상 활동에 몰두하고 있는 동안, 우리 자신의 모습과 자세가 어떠한지를 시각적으로 마주하고 경험해볼 수 있는 기회를 제공합니다.
